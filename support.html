Support

Support for Bendit.

Found a bug, missing a file transformation feature, or want to share a nasty Excel/CSV case? Email us directly — no account needed.

Email support

Best for bugs, feature requests, file import issues and reproducible transformation problems.

Email support@bend-it.net

Useful details to include

  • What file type you used: Excel, CSV or Parquet
  • Approximate file size and number of rows
  • What you expected Bendit to do
  • What happened instead
  • A small anonymized sample file, if possible
